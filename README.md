# Malaysian-Amateur-Radio-Callsign-Search


## Setup Instructions

1. **Create and Navigate to Project Directory**
   ```bash
   mkdir callsign-query
   cd callsign-query
   ```

2. **Initialize a New Node.js Project**
   ```bash
   npm init -y
   ```

3. **Install Required Dependencies**
   ```bash
   npm install express node-fetch
   ```

4. **Create Project Files**
   - **Create `server.mjs`**: 
     ```bash
     touch server.mjs
     ```
   - **Update `package.json`**: Ensure it includes `"type": "module"`.
     ```bash
     nano package.json
     ```

5. **Set Up Frontend Files**
   - **Create `index.html`**: 
     ```bash
     mkdir public
     cd public
     touch index.html
     ```
   - **Create `styles.css`**:
     ```bash
     touch styles.css
     ```
   - **Create `script.js`**:
     ```bash
     touch script.js
     ```

6. **Start the Server**
   ```bash
   npm start
   ```

Your Node.js server will now be running and accessible at `http://localhost:3000`.
